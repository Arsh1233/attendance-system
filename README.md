# AI-Powered Attendance System

An automated attendance system using face recognition technology built with Streamlit and OpenCV.

## Features

- Upload individual student photos
- Upload class/group photos
- Automatic face detection and recognition
- Attendance marking with visual annotations
- Download attendance reports (CSV)
- Download annotated images
- Simple and intuitive web interface
